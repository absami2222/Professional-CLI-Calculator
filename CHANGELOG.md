# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-02-26

### Added
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Continuous number input support
- Division-by-zero error handling
- Scientific operations: power, square root, logarithm (log10)
- Trigonometric functions: sin, cos, tan (degree input)
- Factorial calculation with input validation
- 2×2 matrix addition
- Smart type-sensitive output formatting
- Comprehensive mathematical error handling
- Clean CLI navigation with main menu support
- Professional command-line user interface

### Features
- Full math.h integration
- Domain validation for all scientific functions
- Real calculator-like behavior and error messages
