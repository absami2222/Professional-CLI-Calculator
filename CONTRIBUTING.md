# Contributing to Professional CLI Calculator

Thank you for your interest in contributing! Here's how you can help:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/Project-Calclator.git`
3. Create a feature branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Test thoroughly: `make clean && make run`
6. Commit with clear messages: `git commit -m "Add feature: description"`
7. Push to your fork and submit a pull request

## Code Style

- Use consistent indentation (4 spaces)
- Follow C89/C99 standards
- Add comments for complex logic
- Keep functions modular and single-purpose

## Reporting Bugs

- Use GitHub Issues with a clear title
- Include steps to reproduce
- Provide expected vs. actual behavior
- Note your OS and compiler version

## Feature Requests

- Open an Issue describing the feature
- Explain the use case
- Discuss implementation approach with maintainers

## Pull Request Process

1. Update README.md if documentation changes
2. Add test cases for new features
3. Ensure `make clean && make` succeeds
4. Request review from maintainers

Thank you for contributing!
